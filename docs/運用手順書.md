# 口コミ管理ツール 運用手順書

## 日常の運用フロー

### 新しい講座を開始する場合

```
1. 管理者でログイン
2. ヘッダーの「管理」をクリック
3. 「+ 講座追加」→ 講座名・説明を入力 → 保存
4. 作成した講座の「口コミ管理」をクリック
5. 「一括追加」→ 口コミを1行1件で貼り付け → 一括追加
6. 受講生用のユーザーを作成（ユーザー管理タブ）
7. 受講生にログイン情報とURL（http://localhost:3000）を共有
```

### 受講生に口コミを使ってもらう場合

受講生に以下を伝えてください：

```
【口コミ投稿の手順】
1. http://localhost:3000 にアクセス
2. お伝えしたメールアドレスとパスワードでログイン
3. 対象の講座をタップ
4. 使いたい口コミの「コピー」ボタンをタップ
5. 口コミが自動でコピーされます
6. 投稿先に貼り付けてください

※ 一度コピーした口コミは他の方には表示されません
※ 同じ口コミが被ることはありません
```

### 口コミが足りなくなった場合

```
1. 管理画面 → 対象講座の「口コミ管理」
2. 「一括追加」で新しい口コミを追加
   または
   「すべてリセット」で使用済みの口コミを再利用可能にする
```

### 講座が終了した場合

```
1. 管理画面 → 対象講座の「削除」
   ※ 関連する口コミもすべて削除されます
```

## ユーザー管理

### 新しいユーザーを追加

```
1. 管理画面 → 「ユーザー管理」タブ
2. 「+ ユーザー追加」
3. 名前・メールアドレス・パスワード・ロールを入力
4. 「追加」をクリック
```

### ロールの違い

| ロール | できること |
|--------|-----------|
| user | 口コミの閲覧・コピー |
| admin | 上記 + 講座管理・口コミ管理・ユーザー管理 |

### ユーザーを削除

```
1. 管理画面 → 「ユーザー管理」タブ
2. 対象ユーザーの「削除」をクリック
   ※ そのユーザーが使用した口コミの記録は残ります
```

## バックアップ

### データベースのバックアップ

```bash
# dataフォルダ内のkuchikomi.dbをコピー
cp data/kuchikomi.db data/kuchikomi_backup_$(date +%Y%m%d).db
```

### データベースの復元

```bash
cp data/kuchikomi_backup_YYYYMMDD.db data/kuchikomi.db
```
